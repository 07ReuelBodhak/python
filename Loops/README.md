# Loops

In python there are three main types of loops :

1. For loop
2. While loop
3. Nested loop

## For Loop

Syntax :

```python
for iterator_var in sequence:
    statement(s)
```

## While Loop

Syntax :

```python
while condition:
    statement
```

## nested loop

Syntax :

```python
for var1 in sequence:
    for var2 in sequence:
        statements
```
